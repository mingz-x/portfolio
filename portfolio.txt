프로젝트 소개 : 

기획 배경 : 
서울 열린데이터광장에서 서울시 오픈 API를 제공해 그 중 관심가는 데이터 중 미술관 데이터를 선택하여 기획하게 되었습니다.
서울열린데이터광장 OPEN API 

프로젝트는 자바스크립트 프레임워크인 next.js를 사용하여 작업했습니다.

프로젝트는 React 프레임워크인 Next.js를 사용하여 작업하였습니다.
Next.js is a React framework for building full-stack web applications.
Next.js는 풀스택 웹 애플리케이션을 빌드하기 위한 React 프레임워크입니다. 

페이지 소개/설명 : 
모바일 앱으로 제작하여 큰 메뉴로는 ‘검색, 메뉴, 홈(메인), 스크랩, 마이페이지’로 크게 5개의 메뉴로 구성했습니다.
 
기능 소개/설명 :
검색 : 검색기능
실시간 검색 기능을 구현하여 사용자(유저)의 편의성을 높임
검색어를 입력해서 나타나는 해당 검색어를 클릭하면 상세페이지로 이동

메뉴 : 페이지 이동
작품 리스트 페이지로 이동
관련 주소로 이동하는데 open api가 존재하지 않는 곳이기 때문에 링크로 연결해 클릭시 새창에서 확인 할 수 있도록 연결함

홈(메인) : 상세페이지이동, list페이지 이동
작품 상세페이지 이동
작품 리스트 페이지로 이동

스크랩 : 상세페이지 안에서 스크랩 버튼
상세 페이지 작품 포스터 우측하단에 스크랩버튼과 좋아요 버튼을 만들어 사용자가 스크랩과 좋아요 버튼을 눌러 마이페이지 안에 있는 좋아요 페이지 또는 스크랩 페이지에서 확인 할 수 있도록 설게
스크랩과 좋아요는 firebase에 저장 될 수 있도록 firbase를 사용하여 작업함

마이페이지 : 로그인, 좋아요, 히스토리, 공지사항
로그인 기능으로 좋아요, 스크랩 같은 기능을 사용할 수 있음
로그인은 nextauth를 사용하여 작업함
좋아요와 히스토리는 각각의 firebase에 저장되어 사용자에게 보여질 수 있도록 작업함
공지사항은 따로 open api가 존재하지 않아 임의로 설계하였으나 삭제 예정.





프로젝트 소개 : 

기획배경 : 
(뉴스)오픈 API를 이용해서 뉴스 페이지를 만들어보고자 기획하게 되었습니다.

프로젝트는 프로그레시브 자바스크립트 프레임워크인 vue.js를 사용하여 개발하였습니다. 

페이지 소개/설명 : 
국내와 해외로 나누어 개발하였고, 국내와 해외라는 큰 카테고리 안에 ‘전체, 정치, 경제, 사회, 문화, 세계, 기술/IT, 연예, 사설’로 나누어 탭버튼을 개발하여 원하는 주제의 기사만 볼 수 있도록 기획하였습니다.

파일 구조
views > homeview.vue
views > aboutview.vue

components > newsarticle.vue
뉴스 관련 작업은 모두 components폴더 안에 newsarticle.vue 파일을 만들어 작업하였습니다.
작업 후 views폴더 안에 있는 homeview.vue와 aboutview.vue 파일에 컴포넌트 파일을 불러와서 화면서 보여질 수 있도록 하였습니다.

기능 소개 : 탭버튼, 기사 클릭 시 상세 페이지 이동,   
탭버튼
기사 주제 관련해서 카테고리를 탭버튼으로 작업.
유저가 원하는 주제의 기사만을 볼 수 있도록 함.
기사 클릭시 상세 페이지 이동
기사 제목과 이미지를 보고 기사를 클릭시 상세 페이지, 즉 관련 기사를 전문을 볼 수 있게 페이지 이동.
페이지 이동시 유저의 편리함을 위해 새창으로 이동할 수 있도록 작업(설계/개발)
